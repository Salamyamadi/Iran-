# Iran-
عملیات برای حک و شناسایی کردن تمامی ایرانی ساکن کل کشورهای دنیا و داخل ایران تا مشخص شود که چه اشخاص قدرتمند و شخصیتهای ی مخالف و دشمن نظام جمهوری اسلامی ایران و رهبر جدید امام آقا سید مجتبی حسینی خامنه فرزند امام شهید آقا سید علی حسینی خامنه هستند که بد از مشخص شدن تعداد افراد مشخص شد و اسناد مشخص کرد در عمل برای سرنگونی نظامام جمهوری اسلامی ایرا
